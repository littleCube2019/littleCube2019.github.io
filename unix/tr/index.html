tr處理文件

tr

translate 的縮寫，可以用於替換字元或刪除輸入資料

EX: test.txt

this is a document  
dog   
cat    
dig

替換字元

EX:
cat test.txt | tr “d” “c”
結果

this is a cocument
cog 
cat 
cig

cat test.txt | tr “\n” " "
結果

this is a document dog cat dig

刪除字元 -d

EX: cat test.txt | tr -d " "

thisisadocument
dog 
cat 
dig
