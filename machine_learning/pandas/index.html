pandas套件介紹 - 表格資料處理好工具

2020-08-20

python中的pandas是專門為矩陣型的表格式資料設計的套件
可以由pip install 安裝

Series 和 DataFrame 物件

這是兩大最常使用的pandas物件，其中又以DataFrame為主要使用物件

Series

就是一種python的dictionary，但不指定key(index)的情況下跟list表現類似

*預設類似list*

import pandas as pd
s1 = pd.Series([1,2,3,4])
print(s1)

會有以下結果

0    1
1    2
2    3
3    4
4    5
dtype: int64

其中第一排是s1的index
第二排則是資料本身

所以可以使用0-4 去取用這五筆資料，在此情況下可以用跟list slice相同的方式去取用這些資料

如: [1:3],[:3],[1:5:2]

但是無法使用-1 等負數作為key

*指定index類似dictionary*

import pandas as pd
s2 = pd.Series([10,20,30,40],index=["今天","明天","後天","大後天"])
print(s2)

會有以下結果

今天     10
明天     20
後天     30
大後天    40
dtype: int64

可以使用s2[“今天”],s1[“後天”] 等方法去取用

當然也可以直接丟入dictionary作為參數，即可直接得到一樣的結果

DataFrame

用來處理多維的table型資料，也是此套件中在大機器學習時代中最常被使用到的物件

以下是各操作簡易介紹

*初始化 -- 使用dictionary*

dic = {"x1":[1,2,3,4,5],"x2":[1,3,5,7,9],"y":[2,4,6,8,10]}

df = pd.DataFrame(dic)
df

結果如下圖:

*改表格表頭 -- 使用class中的rename*

df = df.rename({"x1":"test","x2":"testtest"},axis=1)
df

結果如下圖:

說明:
1.第一個參數是吃一個{替代前:替代後}的dictionary，會在指定axis中尋找符合"替代前"的名字並進行替換，若找不到則不執行

2.第二個參數指定要改的表頭，axis=0為row , axis=1為column
(但注意改column看起來會像把第一個row改掉)

所以當然也可以用 df = df.rename({0:5,1:6},axis=0) 去改動index的表頭 (注意index預設是整數型態，因此"0"改不到任何東西)
