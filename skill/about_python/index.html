題目: 判斷數字是否符合條件
小題醒:

  1. range() 與 list slice 一樣 前閉後開
    如range(1,n+1) 才是 [1,n+1]
  2. 注意class 中呼叫自己函式要self.method
    創造自己函式時 傳入參數要有self def f(self , …):
  3. 注意 range(2,2)是不會有東西的!!!
    質數檢查要 range(2,int(num**0.5)+1)
  4. 注意python常常會轉成float
    相除就要記的int(p/n)
