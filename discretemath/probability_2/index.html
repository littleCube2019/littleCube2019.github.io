機率複習(一) : 離散機率(下)

2020-08-23

重點1: Random Variable

將事件本身當作變數，將事件映射的數值,方便我們計算時使用

比如X=第一顆骰子擲出的點數 , Y=第二顆骰子擲出的點數
P(X+Y=6)就可以表達兩次骰子點數和為6的機率

Example 1

有兩個保險人,今天A出事的機率為0.05,B出事的機率為0.1,兩者出事為獨立事件 請討論支付理賠金的機率分布

令X為支付保險人數
P(X=0) = P(A沒出事 $\cap$ B沒出事) = 0.95 * 0.9 = 0.855
P(X=1) = P(A沒出事 $\cap$ B出事) + P(A出事 $\cap$ B沒出事) = 0.95 * 0.1 + 0.05 * 0.9 = 0.14
P(X=2) = P(A出事 $\cap$ B出事) = 0.05 * 0.1 = 0.005

o.w (P(0>X),P(X>3)) = 0

重點2: 機率質量函數 (p.m.f)

如果一個隨機變數發生的可能值皆為countable number，則該隨機變數稱為離散隨機變數

對於離散隨機變數，可以定義probability mass function

Def: probability mass function

P(a) = P{ X=a }
並且有以下性質
(1) X的可能值編號為 $ x_1 , x_2 … x_n $
P(X=$x_i$) > 0 if i $\in$ [1,n]
P(X)=0 else

$(2) \sum_{i=0}^{\infty}P(X=x_i)=1 $
總機率和為1

Example 2

給定X的pmf為$$P(i) = \frac{c\lambda^i}{i!}$$ i=0,1,2,3… , $\lambda$為一正數
求(1) P{X=0} (2) P{X>2}
Sol:
(1) P{X=0} = $\frac{c\lambda^0}{0!}$ = c

$\sum_{i=0}^{\infty}P(X=x_i)= ce^{\lambda}=1$
又依據泰勒展開式，$e^x= \sum_{i=0}^{\infty}x^i/i!$
故 c = $e^{-\lambda}$

故答案為$e^{-\lambda}$

(2) P{X>2} = 1-P{x=0}-P{x=1}-P{X=2} =
1-$e^{-\lambda}$-$e^{-\lambda}$*$\lambda$-$\frac{e^{-\lambda}*\lambda^2}{2}$

Reference

A First Course in Probability 9ed
